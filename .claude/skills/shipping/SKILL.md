# /shipping — Доставка через СДЭК

## Алгоритм

1. Вызови get_cities чтобы найти коды городов отправления и назначения
2. Вызови calculate_tariff с тарифом 136 для расчёта стоимости
3. Вызови list_pvz для обоих городов чтобы показать ближайшие ПВЗ
4. Покажи результат: стоимость, сроки, ПВЗ

## Примеры

    /shipping Москва Казань 2
    /shipping Новосибирск Екатеринбург 5
